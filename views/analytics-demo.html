üöÄ Analytics Implementation

For Frontend Applications

If you're building a frontend application that consumes this API, integrate Vercel Web Analytics based on your framework:

  • React/Create React App: Import and use the <Analytics /> component from @vercel/analytics/react
  • Next.js: Import from @vercel/analytics/next and add to your layout
  • Vue: Import from @vercel/analytics/vue and add to your main component
  • Vanilla JS: Use the inject() function from @vercel/analytics
  • HTML Sites: Add the analytics script directly to your HTML files

Example: React Application

// Install the package
npm install @vercel/analytics

// In your App.jsx or App.tsx
import { Analytics } from '@vercel/analytics/react';

export default function App() {
  return (
    <div>
      {/* Your app content */}
      <Analytics />
    </div>
  );
}

Example: Vanilla JavaScript

// Install the package
npm install @vercel/analytics

// In your main JavaScript file
import { inject } from '@vercel/analytics';

inject();
